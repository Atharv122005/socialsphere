# Overview

SocialSphere is a frontend-focused social media platform developed using React and Vite.  
The application is designed with reusable components, responsive layouts, and modern UI patterns to simulate a real-world social networking experience.

The project focuses on:

- Component-based architecture
- Clean frontend structure
- Interactive user interface
- Responsive design principles
- Modern React development workflow

# Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Atharv122005/socialsphere.git
```

Move to the project directory:

```bash
cd socialsphere
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open browser:

```bash
http://localhost:5173
```
